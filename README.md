# Biblioteca de Llibres — Ainhoa Alvarez

## Descripció

Aplicació Angular per gestionar una biblioteca personal de llibres.
Permetrà visualitzar, afegir i organitzar llibres durant el curs.

## Stack tècnic

* Angular 18
* Arquitectura Standalone
* Routing habilitat
* Estils SCSS

## Estat

Configuració inicial de l'entorno completada i verificada.
